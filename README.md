# Interest-Based E-Commerce Recommendation System

## 📌 Project Overview
This project is a web-based e-commerce application developed to analyze user browsing behaviour and predict purchase intention. The system provides personalized product recommendations based on user interaction patterns, helping improve user experience and engagement.

The platform simulates real-world online shopping workflow including product browsing, user activity tracking and recommendation generation.

## 🚀 Key Features
- User login and session handling
- Product browsing and category filtering
- Behaviour tracking based on user interactions
- Personalized product recommendation logic
- Structured product data storage and retrieval
- Interactive and responsive user interface

## ▶️ How to Run the Project
1. Clone or download the repository
2. Install Node.js dependencies using `npm install`
3. Configure database connection settings
4. Start server using `node app.js`
5. Open browser and access the application via localhost
